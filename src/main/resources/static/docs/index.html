Users

회원가입

설명

  • 이메일/비밀번호/이름으로 회원가입합니다.

  • 비밀번호는 서버에서 BCrypt로 암호화되어 저장됩니다.

Request

POST /api/users HTTP/1.1
Content-Type: application/json
Content-Length: 113
Host: localhost:8080

{"password":"password1234","name":"테스트유저","email":"test+0f1c498e-0892-4455-983a-44716b2eac39@test.com"}
