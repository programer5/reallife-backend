Note

SSE 재연결(replay)을 위해 클라이언트는 Last-Event-ID 헤더를 보낼 수 있습니다.

  • Last-Event-ID 값은 "마지막으로 수신한 SSE 이벤트의 id" 입니다.

  • 본 프로젝트는 이벤트 id를 다음 규칙으로 통일합니다.

  • message-created / message-deleted: msg:{messageId}

  • notification-created: noti:{notificationId}
