Users

회원가입

설명

  • 이메일/비밀번호/이름으로 회원가입합니다.

  • 비밀번호는 서버에서 BCrypt로 암호화되어 저장됩니다.

Request

POST /api/users HTTP/1.1
Content-Type: application/json
Content-Length: 113
Host: localhost:8080

{"password":"password1234","name":"테스트유저","email":"test+68b9391d-dfdb-42d1-ac2e-5b45027abb40@test.com"}
Path Type Description

email

String

사용자 이메일(로그인 ID)

password

String

비밀번호(최소 8자). 서버에 BCrypt로 암호화되어 저장됨

name

String

사용자 이름
