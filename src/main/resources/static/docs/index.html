Users

회원가입

설명

  • 이메일/비밀번호/이름으로 회원가입합니다.

  • 비밀번호는 서버에서 BCrypt로 암호화되어 저장됩니다.

Request

POST /api/users HTTP/1.1
Content-Type: application/json
Content-Length: 113
Host: localhost:8080

{"password":"password1234","name":"테스트유저","email":"test+8b88ea75-821b-4eb8-8137-6f6c80d5456f@test.com"}
Path Type Description

email

String

사용자 이메일(로그인 ID)

password

String

비밀번호(최소 8자). 서버에 BCrypt로 암호화되어 저장됨

name

String

사용자 이름

Response

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 173

{"id":"5f22553e-d004-4d4b-b858-1422c116ecfe","email":"test+8b88ea75-821b-4eb8-8137-6f6c80d5456f@test.com","name":"테스트유저","createdAt":"2026-01-28T13:29:11.0958955"}
Path Type Description

id

String

사용자 식별자(UUID)

email

String

사용자 이메일

name

String

사용자 이름

createdAt

String
