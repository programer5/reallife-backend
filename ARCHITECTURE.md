# ARCHITECTURE.md

RealLife 서비스 전체 아키텍처 문서

------------------------------------------------------------------------

# 1. 서비스 개요

RealLife는 단순 SNS가 아니라

    피드 → 댓글 → Action → 대화 → 실제 삶

흐름을 만드는 플랫폼이다.

------------------------------------------------------------------------

# 2. 시스템 아키텍처

    Frontend (Vue3 + Vite)
            ↓
    Backend (Spring Boot)
            ↓
    MySQL + Redis
            ↓
    SSE (Realtime)

------------------------------------------------------------------------

## Action (Pin)

RealLife 핵심 기능

    댓글
    ↓
    Action 생성
    ↓
    Conversation Dock
    ↓
    Action Timeline
    ↓
    Reminder

------------------------------------------------------------------------

## Conversation

-   채팅
-   메시지 전송
-   unread 관리
-   Action Dock
